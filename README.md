# Translator Chatbot

A simple translator chatbot where users can select source and target languages, enter text, and get real-time translations.  

---

## Tech Stack
- **Frontend:** React, Vite, Bootstrap  
- **Backend:** Node.js, Express  
- **API:** Translation API (Azure)
